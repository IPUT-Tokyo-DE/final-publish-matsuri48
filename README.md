# 監視下迷路

## . ゲームの基本コンセプト
プレイヤーは、レベルごとに広大で複雑になっていく迷路に閉じ込められます。制限時間内に迷路のどこかに配置された「ゴール」をすべて見つけ出し、脱出することが目的です。

2. 独自システム：「光と闇」のサイクル
このゲーム最大の特徴は、一定時間ごとに訪れる「フェーズの切り替え」です。
明るい時間（Bright Time）: 迷路全体が見渡せる安全な時間です。この間に、壁の配置やゴールの方向を必死に覚える必要があります。
暗い時間（Dark Time）: 画面が漆黒に包まれます。自分の記憶だけを頼りに、一歩一歩手探りで進まなければなりません。

3. 心理的な恐怖演出：「監視」
暗闇フェーズになると、画面には**「EYES ON YOU（お前を見ているぞ）」**という赤い文字が浮かび上がります。 視界が制限されるだけでなく、「何かに見られている」という心理的なプレッシャーがプレイヤーを襲い、パニックを誘います。

4. 段階的な難易度アップ
レベルが上がるにつれて、以下の要素が組み合わさり、絶妙な手応えを感じさせます。
迷路の巨大化: 15x11から始まり、どんどん複雑な構造に進化します。
ゴールの増加: 探すべき目標が増え、より深い探索が求められます。
時間の短縮: 制限時間と明るい時間の両方が厳しくなり、焦りが生まれます。

視界を奪われる恐怖の中で、自分の記憶だけを信じて出口を探す、心理的迷路サバイバル

## 操作説明

暗い時は動いてください
明るい時にはひたすら迷路を暗記してください

矢印キーでプレイヤーを動かせます
STARTボタンでゲーム開始
RETRYボタンでゲームを再開できます
